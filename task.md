## 总体定位：电磁学物理沙盒《场》

**核心体验**：像玩积木一样搭建电磁学装置——用带电小球拼出电场迷宫，用磁铁和导线造出发电站，用电路元件组装逻辑门，最后甚至可以搭建一个简易的无线电收发系统。

**设计原则**：
- **所见即所算**：所有现象由底层物理引擎实时计算，不预设脚本。
- **元件即语言**：每个阶段添加一批新“玩具”，玩家自由组合。
- **工具化可视化**：提供电场线、磁场线、电势表、示波器等观测工具，沙盒的核心是“看见看不见的东西”。

---

## 阶段一：静电沙盒

**目标**：建立物理沙盒的核心框架——空间、物体、力、可视化层。验证点电荷库仑力模拟。

### 核心引擎能力
- 点电荷电场叠加（库仑定律）
- 导体表面电荷分布（尖端放电可后期加）
- 电场可视化（电场线、等势面）
- 刚体力学 + 库仑力耦合

### 沙盒元件库
| 类别 | 元件 | 说明 |
|------|------|------|
| 电荷源 | 正/负点电荷球 | 可固定或自由，电量可调 |
| 电荷源 | 带电平板 | 产生近似匀强电场 |
| 导体 | 金属球/块 | 感应起电，电荷自动分布 |
| 绝缘体 | 塑料块/线 | 可摩擦起电 |
| 探针 | 电场探针 | 拖动显示所在点E的大小方向 |
| 探针 | 电势探针 | 显示电势值 |
| 工具 | 电荷注入器 | 给物体充上指定电量 |

### 预置场景/实验模板
- **静电摆**：两个同号电荷球用细线悬挂，观察库仑斥力导致的偏转
- **电荷飞船**：给平台和底座带上同号电荷，让它悬浮
- **静电除尘模型**：带电网格吸附飘散的轻小颗粒
- **电场迷宫**：布置电荷阵列，用探针找出电势为0的路径

### 玩家可搭建的玩法
- 设计“静电陷阱”困住自由电荷球
- 搭建静电悬浮装置
- 挑战：只用库仑力让小球走完指定路径

## 阶段二：磁场沙盒

**目标**：引入磁场、磁铁、运动电荷的洛伦兹力。电场和磁场并行，但暂时不耦合变化。

### 核心引擎能力
- 永磁体磁场（偶极子模型）
- 电磁铁磁场（电流→磁场）
- 洛伦兹力 F=qv×B
- 磁场可视化（磁感线、磁通量密度显示）

### 沙盒元件库（在阶段一基础上新增）
| 类别 | 元件 | 说明 |
|------|------|------|
| 磁源 | 条形/蹄形永磁铁 | 固定磁场，强度可调 |
| 磁源 | 电磁铁 | 通入电流产生磁场，可开关 |
| 磁源 | 亥姆霍兹线圈对 | 产生均匀磁场区 |
| 导磁 | 铁芯/磁轭 | 导磁，可改变磁路 |
| 探针 | 磁探针 | 显示B的大小方向 |
| 工具 | 磁化/消磁器 | 磁化或消磁物体 |

### 预置场景/实验模板
- **电子在磁场中的圆周运动**：发射带电粒子进入均匀磁场，观察圆形轨迹
- **速度选择器**：交叉的E场和B场，只有特定速度的粒子能直线通过
- **磁悬浮陀螺**：环形磁铁阵列悬浮一个旋转磁体
- **阴极射线管模型**：电场加速+磁场偏转，粒子打到荧光屏

### 玩家可搭建的玩法
- 搭建磁力炮：用多个电磁铁依次开关加速铁磁弹丸
- 设计磁场迷宫让带电粒子走特定路径
- 用磁铁阵列创造“磁阱”捕获运动电荷

---

基于你已有的“静电沙盒”和“磁场沙盒”框架，下一步自然是从“静”走向“动”，从“分立”走向“统一”。为了体现麦克斯韦方程组的核心物理思想——**变化的电场产生磁场，变化的磁场产生电场，并最终指向电磁波**，我为你设计了两个新的扩写阶段。这两个阶段刻意回避了“导线电路”等具象描述，转而使用更符合“场”沙盒气质的抽象场源和通量元件。

---

## 阶段三：时变场沙盒

**目标**：引入变化的场，实现法拉第电磁感应定律与安培-麦克斯韦定律。电场和磁场开始“对话”，但暂不形成向外传播的自由电磁波（近场区）。

### 核心引擎能力
- **法拉第电磁感应**：时变磁场 → 感生电场（涡旋电场），感生电场线呈闭合涡旋状，不依赖导体存在
- **安培-麦克斯韦定律**：时变电场（位移电流） → 感生磁场，即使没有自由电荷运动也能激发磁场
- **场-场直接耦合**：时变E场实时驱动B场变化，时变B场实时驱动E场变化（近场耦合）
- **涡旋场可视化**：专门着色器显示闭合的感生电场线/感生磁场线，与静场的“发散状”线系区分

### 沙盒元件库（在阶段一+二基础上新增）

| 类别 | 元件 | 说明 |
|------|------|------|
| 时变场源 | 交变电偶极子 | 两个异号电荷，间距远小于波长，电量按正弦/自定义波形变化 |
| 时变场源 | 交变磁偶极子（振荡小磁铁或振荡电流环） | 磁矩按正弦/自定义波形变化 |
| 时变场源 | 匀强时变电场区域 | 平板区域内E场大小按波形变化，方向固定 |
| 时变场源 | 匀强时变磁场区域 | 螺线管/亥姆霍兹区域内B场大小按波形变化，方向固定 |
| 无源场感应体 | 涡流环（非导体） | 放置在时变磁场中，会自发产生闭合的感生电场线，可加速内部自由电荷 |
| 无源场感应体 | 极化涡旋盘 | 放置在时变电场中，盘内等效位移电流产生感生磁场线 |
| 探针/仪表 | 感生场探针 | 区分“库仑场”与“感生场”两个分量，分别显示大小方向 |
| 探针/仪表 | 位移电流密度探针 | 显示空间中 ∂E/∂t 的大小和方向 |
| 探针/仪表 | 磁通变化率探针 | 显示通过任意用户定义曲面的 dΦ_B/dt |
| 波形发生器 | 函数场发生器 | 可附加到任意时变场源上，提供正弦、方波、三角波、自定义时间序列 |

### 预置场景/实验模板

1.  **无导线的“感应加速环”**
    - **配置**：中央一个时变磁偶极子（磁场垂直纸面向外，正弦变化）。周围空间中空无一物，没有导体环。
    - **现象**：观察感生电场线——它们是一圈圈围绕磁偶极子轴的同心闭合环。将一个自由电荷放在任一环上，电荷会自动开始沿环线加速绕圈。改变磁场变化率，加速力度随之改变。
    - **点拨**：电场不需要导线就能存在并做功。这是“涡旋电场”的本征形态。

2.  **位移电流“磁力镜”**
    - **配置**：两个平行平板（不连接任何电源），分别带有时变电量（例如一板电荷+q·sinωt，另一板-q·sinωt）。两板之间是真空。
    - **现象**：在两板间隙处放置一个磁探针。即使没有任何自由电荷流动（板间真空），探针仍能测到感生磁场线——这些磁感线环绕着板间变化的电场方向。磁场强度正比于 dE/dt。
    - **点拨**：变化电场本身就是磁场的源。麦克斯韦将这一项加入后，方程组才有了对称性。

3.  **场耦合谐振器（近场能量无线传输）**
    - **配置**：左侧放置一个交变电偶极子（发射端，频率f）。右侧一定距离外放置一个交变磁偶极子（接收端），二者之间没有导线、没有磁芯。
    - **现象**：发射端的时变E场会在空间中感生出时变B场（位移电流项），该时变B场到达接收端后，通过法拉第感应驱动接收端的磁矩振荡。调整距离和方向，接收端振荡幅度随发射端波形同步变化。
    - **扩展**：玩家可加入第三、第四个偶极子，组成近场耦合网络。能量在“看不见”的场之间跳跃。
    - **点拨**：这是变压器原理的“场版本”——不用铁芯，不用线圈，只靠时变E和时变B在真空中的相互激发传递能量。

4.  **感生电场驱动物流**
    - **配置**：一条由“时变磁场区域”元件排列成的环形轨道（各区域的B场按时间相位差变化，类似行波磁场）。轨道上方放置自由电荷小球。
    - **现象**：小球不接触任何导体，仅被感生电场推着沿轨道行走。改变各区域B场的时序，小球加速或减速。
    - **点拨**：这对应粒子加速器中“感应加速”的原理，完全基于法拉第定律。

### 玩家可搭建的玩法

- **无接触传送带**：用一串时变磁偶极子，按相位顺序振荡，推动一串带电颗粒沿直线运动
- **场耦合逻辑门**：利用两个交变电偶极子的近场干涉，在特定空间点产生“场强和”超过阈值或低于阈值的区域，以此作为“场逻辑门”的输入输出（例如：两个源都有场→某点感生磁场强度>1→触发一个磁控开关）
- **时变场透镜**：布置介电常数渐变的材料（阶段三后期可引入简单线性介质），让时变电场线的弯曲，聚焦到远处一个接收偶极子上
- **反法拉第挑战**：给定一个空间区域内的时变磁场分布，要求玩家放置若干被动“极化涡旋盘”（不主动供电），使得这些盘产生的感生磁场恰好抵消原始磁场在某个关键点的变化率——实现局部“磁通锁定”

## 阶段四：电磁波沙盒

**目标**：彻底解放场。时变场从“近场耦合”过渡到“远场辐射”，电场和磁场在空间中互相支持、脱离源向外传播。完整展示麦克斯韦方程组预言的电磁波。

### 核心引擎能力
- **波动方程求解**：空间离散化后，直接求解 ∂²E/∂t² = c² ∇²E 和类似形式的B场（或使用FDTD引擎）
- **辐射边界**：模拟无限大空间，波到达边界不被反射（或用户可选择周期性边界/反射边界）
- **极化与模式**：支持线极化、圆极化、椭圆极化波，支持TEM、TE、TM模（在波导/开放空间）
- **波动物体交互**：电磁波遇到介质界面→反射、折射、全反射；遇到导体→趋肤效应、反射；遇到狭缝/障碍物→衍射、干涉
- **波速与色散**：真空中波速恒定；可引入色散介质，不同频率波速不同
- **能流可视化**：实时绘制坡印廷矢量（S = E×B）的分布，显示能量流动方向与密度

### 沙盒元件库（在阶段一+二+三基础上新增）

| 类别 | 元件 | 说明 |
|------|------|------|
| 辐射源 | 偶极子天线（抽象场型） | 交变电偶极子工作在高频（波长可比拟尺寸），辐射球面波，有方向图 |
| 辐射源 | 半波振子阵列 | 多个偶极子相位排列，可形成定向波束 |
| 辐射源 | 开波导口 | 内部时变场从开口辐射到外部空间 |
| 辐射源 | 平面波入射源 | 从边界某方向射入均匀平面波（模拟无限远信号源） |
| 被动场元件 | 反射屏（理想导体） | 电磁波完全反射，节点处切向E=0，形成驻波 |
| 被动场元件 | 介质板 | 设置相对介电常数ε_r，波速减慢、波长缩短 |
| 被动场元件 | 铁磁/铁电板 | 设置μ_r，改变波阻抗 |
| 被动场元件 | 极化栅 | 只允许特定极化方向的波通过 |
| 探针/仪表 | 波动探针 | 显示某点E/B随时间波形，可测量振幅、相位、频率 |
| 探针/仪表 | 坡印廷矢量流线 | 动态线条动画，显示能量流动路径 |
| 探针/仪表 | 驻波比计 | 显示空间中的波腹/波节点位置 |
| 工具 | 频率扫描仪 | 扫描空间场对频率的响应（类似频谱仪） |

### 预置场景/实验模板

1.  **偶极子天线辐射与坡印廷流**
    - **配置**：中心一个偶极子天线源（频率f）。周围为空旷空间。
    - **现象**：近场区（波长的若干分之一内）感生场和感应场互相交缠，坡印廷矢量方向复杂甚至有回流。远场区（数倍波长外）电场和磁场垂直正交，坡印廷矢量均匀向外辐射。播放动画：能量像“气泡”一样从天线表面成层剥落、向外扩散。
    - **点拨**：电磁波就是“场脱离了源，独立在空间中旅行”。

2.  **双缝干涉与偏振验证**
    - **配置**：一个平面波源垂直照射到一块带两条狭缝的导体屏上，屏后是观测屏（可用场探针阵列代替）。
    - **现象**：屏后出现明暗相间的干涉条纹。改变入射波的极化方向（例如垂直缝方向 vs 平行缝方向），条纹对比度变化——偏振方向平行于狭缝时衍射更强。
    - **扩展**：换成圆极化波，观察干涉图样的旋转特性。
    - **点拨**：干涉是波动性的直接体现，无需“导线”或“电荷”，只有E和B的相位叠加。

3.  **介质波导与全反射**
    - **配置**：一条高ε_r介质条带（例如ε_r=9）悬浮在真空中，一端放置偶极子天线馈入电磁波。
    - **现象**：波在介质条带内部沿条带方向传播，在条带边缘发生全内反射，大部分能量被约束在条带内部及附近。若条带弯曲，波仍能“拐弯”跟随条带。
    - **点拨**：这是光纤/介质波导的原理——光/电磁波被“困”在高折射率介质中行走。

4.  **谐振腔模式**
    - **配置**：一个金属立方体空腔（理想导体边界），内部是真空。放置一个短脉冲偶极子源激发宽频场。
    - **现象**：腔内场逐渐组织成特定的驻波模式。用频率扫描仪可看到多个尖峰频率（本征模）。玩家可以改变腔体尺寸，观察模式频率的变化。
    - **点拨**：谐振腔相当于电磁波的“乐器”，不同尺寸有不同的“音调”。

5.  **电磁波推动物体——光压演示**
    - **配置**：一个强平面波源（高功率）照射一块悬浮的超薄导体箔片（质量极小）。
    - **现象**：箔片被波推动而缓慢加速，方向与波传播方向一致。
    - **点拨**：电磁波携带动量，其宏观表现即是光压——坡印廷矢量除以c²给出动量密度。

### 玩家可搭建的玩法

- **全息干涉仪**：搭建一个马赫-曾德尔干涉仪结构（两个反射屏，两个分束介质板），将一束平面波分成两路再合束，观察干涉条纹。改变其中一路的介质板厚度（即光程），条纹移动——玩家可据此测量波长。
- **无线能量灯塔**：用偶极子阵列合成一个高度定向的波束，照射到远处一个“接收天线元件”（阶段四新增的简单接收器，会将入射波的功率转换成某种可视化效果，比如发光强度），目标是让接收器最亮。
- **极化加密通信**：发射端交替发送水平极化和垂直极化波，接收端放置一个极化栅，只有正确设置栅的方向才能收到信号。玩家可以模拟“极化密钥”的简单概念。
- **隐形斗篷（简易版）**：用多层介质板围绕一个物体，调整各层的ε_r和厚度，使得从外部特定方向入射的平面波经过物体周围后，出射波的波前与无物体时几乎相同。用平面波源照射，在背后用探针阵列观测场分布——物体似乎“消失”了。
- **黑洞模拟**：布置一个径向变化的介电常数分布（外高内低，特定形式），使得电磁波从外部进入后无法逃逸，坡印廷矢量流线全部指向中心。这虽然不严格对应引力黑洞，但在“场”沙盒里可以成为有趣的谜题。
